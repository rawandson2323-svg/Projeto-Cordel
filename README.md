Projeto Cordel Moderno
Este é um projeto desenvolvido durante o curso de HTML5 e CSS3 do canal Curso em Vídeo, ministrado pelo professor Gustavo Guanabara.

O objetivo do projeto foi aplicar técnicas avançadas de posicionamento de imagens e o efeito Parallax, onde o fundo permanece fixo enquanto o conteúdo textual desliza sobre ele.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização, efeitos de sombreamento, transparências e o efeito de fundo fixo.

VS Code: Editor de código utilizado no desenvolvimento.

📱 Funcionalidades
Efeito Parallax: Implementado através da propriedade background-attachment: fixed.

Design Responsivo: O site adapta o tamanho das fontes e imagens conforme a largura da tela.

Identidade Visual: Uso de fontes customizadas para remeter à literatura de cordel.
